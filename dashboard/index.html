HQ

Overview

n8n is an open-source, low-code workflow automation platform focused on developers and technical teams. Its execution-based pricing (whole workflow = 1 execution) provides up to 75% cost savings vs task-based competitors. Includes 70+ AI nodes with LangChain integration, vector databases, and memory management.

Key Strengths

  • Free self-hosted option with unlimited workflows
  • Custom JavaScript/Python in any node
  • 70+ AI nodes (LangChain, vector stores)
  • Full data sovereignty and GDPR compliance
  • 75% cost savings vs Zapier at high volume
